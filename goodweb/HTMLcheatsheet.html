Cheatsheet

Command Prompt

Doe cmd om in je command prompt te komen.

Doe cd om in een anderen directories.

Doe cd .. om terug te gaan in je directories.

Doe dir om te kunnen zien wat in je map staat.

Doe dir*txt als je alleen de txt bestanden wil zien.

Rmdir is om je directories te verwijderen.

Rmdir /s is voor als het vol staat met bestanden.

Git init

Doe git status om te checken of er niet al een git in je map staat.

Doe git init (ALLEEN IN DE MAP DIE JE MOET GEBRUIKEN).

Doe git status om te checken of het gelukt is.

Doe git add (je bestand naam) om het bestand te add aan je git (je kan ook git add . doen om allen bestanden in 1 keer te add).

Doe git status om te checken om of hij goed is added.

Doe git commit -m “(welke naam je ook wil)”.

doe git log om te zien of het gecommit is.

Git push naar github

Maak repository in je github aan.

Doe git remote add origin (je link van je github).

Doe git push -u origin Main.

Doe git log om te zien of je het goed heb gedaan.

Doe git push.

Doe git pull om een bestand van github af te halen.

Doe git clone om een github te clonen en in jouw bestanden te kunnen zetten.
